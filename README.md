# 🚀 Retrieval-Augmented Generation (RAG) Learning Repository

Welcome to my RAG learning repository! This repository documents my journey of understanding, implementing, and experimenting with **Retrieval-Augmented Generation (RAG)** systems—from the fundamentals to advanced **Agentic RAG** architectures.

The goal is not just to build working applications, but to deeply understand every component involved in modern RAG systems, including data ingestion, chunking, embeddings, retrieval, vector databases, LLM integration, and autonomous agents.

---

## 📖 What is RAG?

Retrieval-Augmented Generation (RAG) combines the power of **Large Language Models (LLMs)** with **external knowledge retrieval systems**. Instead of relying solely on the model's pre-trained knowledge, RAG retrieves relevant information from custom data sources and provides it as context to the LLM, resulting in more accurate, up-to-date, and grounded responses.
